AULA 3  - O QUE SÃO BANCOS DE DADOS

O QUE SAO DADOS: DADOS SÃO UM FATO, PRECISAM SER LAPIDADOS PARA SEREM TRANSFORAMADOS EM INFORMAÇÕES


DIFERENCA ENTRE BD - SGBD

BD: Um banco de dados é uma coleção organizada de informações - ou dados - estruturadas, normalmente armazenadas eletronicamente em um sistema de computador.
   
AULA 4: SGBDs

SGBD: SISTEMA DE GERENCIAMENTO DE BANCO DE DADOS

SOFTWARE DE PROPOSITO GERAL, GERENCIAR OS DADOS QUE ESTÃO DENTRO DO SGBD

	MANIPULAÇÃO -> RELACIONADA A RECUPERAÇÃO DE INFORMAÇÃO E CRIAÇÃO DE RELATORIOS. 
	COMPARTILHAMENTO -> FORNECE ACESSO SIMULTANEOS MAS VOCE PRECISA GERENCIAR OS ACESSOS
	PROTEÇÃO E CICLO DE VIDA DE LONGO PRAZO
	
METADADOS: INFORMAÇÕES QUE FORNECEM UMA DESCRIÇÃO CONCISA DOS DADOS CONTIDOS NO BDW
CONSTRUÇÃO
MANIPULAÇÃO: OS USUARIOS CRIAM QUERYS PARA MANIPULAR OS DADOS TRANZENDO INFORMAÇÕES 
COMPARTILHAMENTO: POR PADRAO O BD REALIZA O BLOQUEIO E A LIBERAÇÃO DAS TABELAS, MAS VOCE PODE LIBERAR E FORNECER O COMPARTILHAMENTO
